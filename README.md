# Student Management System

A console-based Student Management System built in C and C++.

## About
This project was built twice — first in C, then rewritten in C++ — to understand the difference between procedural and object-oriented programming.

## Features
- Add student records (name, roll number, marks)
- Auto calculates grade based on marks
- View all students in a table
- File handling to save and load data

## Files
- `students.c` — C version
- `students_v2.cpp` — C++ version

## Tech Stack
- Language: C, C++
- Compiler: Clang (macOS)
